# picgo-plugin-multi-uploader

🚀 **PicGo 插件**：让你一次上传图片到多个图床！支持并行上传、自动重试、统一文件名、上传结果 Markdown 汇总输出。

---

## ✨ 功能特性
- ⚡ 并行上传到多个图床（速度快）
- 🔁 上传失败自动重试（可配置次数与延迟）
- 🧱 所有图床保持相同文件名
- 🧩 上传完成后生成 Markdown 链接汇总（直接复制粘贴）
- 🪶 兼容所有 PicGo 官方和第三方图床

---

## 📦 安装

```bash
npm install picgo-plugin-multi-uploader -g
# 或者
picgo install picgo-plugin-multi-uploader
```
